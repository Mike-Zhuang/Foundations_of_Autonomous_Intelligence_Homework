# Foundations of Autonomous Intelligence (自主智能基础)

欢迎来到《自主智能基础》课程的作业与实践代码库。本仓库记录了在算法的思考、数据分析处理以及复杂动态系统演化过程中的思考和工程化实现。

以下为当前目录下包含的核心作业/项目模块，所有项目均采用标准化、高内聚结构开发，并配备详尽的可视化手段与使用文档。点击各个子项目链接了解更多详情及操作方法。

## 📁 课程项目导航

| 项目名称 | 描述 |
| :--- | :--- |
| **[1. 教学管理系统程序开发算法迁移实践](./1.%20教学管理系统程序开发算法迁移实践/README.md)** | 基于 Python OOP 面向对象实现的学生成绩管理与分析系统。实现了信息增删、GPA换算、并生成丰富的数据可视化雷达图与直方图。 |
| **[2. WIM（动态承重系统）数据分析](./2.%20WIM（动态承重系统）数据分析/README.md)** | WIM（Weigh-In-Motion）系统采集的交通与载重数据分析程序。包含多维度车流清洗、特征分析规律与海量数据统计图表生成。 |
| **[3. 自然界、生活、工程领域中的动态演变算法及实现](./3.%20阶段1实践：自然界、生活、工程领域中的动态演变算法及实现/README.md)** | **🔥 核心大项目：LBM D2Q9 流体力学仿真**<br>一款支持实时交互的高性能二维计算流体力学渲染引擎，可以交互式模拟与验证卡门涡街、BFS迷宫寻路、以及 NACA 系列空气动力学特性。 |

> 以上工程均针对 Mac/Linux/Windows 环境完全兼容，依赖 `numpy` 及 `matplotlib`。详细的运行方法、环境搭建以及架构原理均在各个子文件夹内的 README完整呈现。
