## 二、基于格子玻尔兹曼方法的多场景流体动态演化仿真系统软件说明书

## 1. 软件概述

本软件名称为“基于格子玻尔兹曼方法的多场景流体动态演化仿真系统”，是一套面向二维流体动力学过程展示、动态演化算法教学和复杂边界流动分析的软件系统。软件以 Python 为开发语言，采用格子玻尔兹曼方法中的 D2Q9 离散速度模型作为核心求解框架，在统一界面中实现速度场、涡量场、涡核区域和升力历史的同步可视化显示。软件既可用于典型圆柱绕流和卡门涡街现象展示，也可用于复杂图像边界建模、迷宫流体寻路演示以及翼型升力分析，具有较强的教学展示价值和工程启发意义。

## 2. 开发背景与目的

在流体动力学课程、智能演化算法课程和工程数值分析实验中，传统静态图示难以完整反映流体在复杂边界条件下的动态变化过程，学习者通常难以直观理解碰撞、迁移、边界反弹、涡量形成、尾迹脱落和升力变化之间的联系。为解决这一问题，本软件以轻量化、可交互、可视化为目标，将二维流体数值模拟与图形界面交互融合，通过较少依赖即可在普通个人计算机上运行，实现从基础物理模型到复杂应用场景的统一仿真与观察。该软件能够帮助使用者更加直观地理解动态演化规律，提高教学演示效率，同时为相关课程设计、竞赛展示和软著成果申报提供清晰的软件化载体。

## 3. 软件总体功能

本软件具备以下几类核心功能：

1. 提供基于 LBM D2Q9 模型的二维流体演化求解能力，可模拟不同雷诺数下的流动状态变化。
2. 提供多种内置障碍物，包括圆形、方形、椭圆、三角形、平板、菱形以及 NACA0012 翼型，用于分析不同几何边界对流场结构的影响。
3. 提供复杂图像障碍导入功能，可将黑白图像转化为流体边界，构建迷宫或复杂通道场景。
4. 提供 BFS 最短路径求解功能，并将路径结果与流体传播过程耦合展示，实现迷宫通路逐步显现的动态效果。
5. 提供速度场、涡量场、Q 准则涡核和升力系数历史曲线的联合可视化显示。
6. 提供鼠标实时绘制障碍、擦除障碍、快捷键调参、快照保存和动画导出等交互功能。

## 4. 软件组成与模块说明

本软件主要由以下模块组成：

### 4.1 流体求解模块

流体求解模块是软件的核心部分，主要对应 [`Simulation.py`](/Users/mike/Documents/University/Grade_1B/classes/Foundations of Autonomous Intelligence/Homework/3. 阶段1实践：自然界、生活、工程领域中的动态演变算法及实现/Simulation.py#L22) 中的 `LBMKarmanSimulation` 类。该模块完成流体分布函数初始化、宏观密度与速度计算、平衡态分布构造、碰撞更新、迁移更新以及入口出口边界处理。程序利用 NumPy 数组广播和矩阵运算替代逐点循环，实现了较高的运行效率。

### 4.2 障碍物建模模块

软件支持多种障碍物生成方式。对于标准几何体，系统可直接构造圆形、方形、椭圆、三角形、平板和菱形边界；对于航空场景，系统内置 NACA0012 翼型公式并支持攻角旋转；对于复杂边界，系统可读取外部图像并转换为二值障碍掩码。该模块使软件具备了较好的场景扩展能力和工程表达能力。

### 4.3 迷宫求解与路径显示模块

当软件导入迷宫图像后，会对障碍区域进行识别，并在可通行区域内执行广度优先搜索算法。系统以左侧入口自由单元作为搜索起点，以右侧出口区域作为目标，求解一条最短路径。求解得到的路径不会一次性全部显示，而是随着流体速度传播逐渐高亮显示，从而形成“算法结果随流场演化被点亮”的动态展示效果，增强了软件的可视化表现力。

### 4.4 气动力分析模块

软件提供三种气动力估算方式，分别为动量交换法、压力积分法和基于环量的升力估算方法。系统对这三类结果进行同步计算，并将升力系数以历史曲线形式展示在右侧图表区域，可用于观察周期性尾迹对气动力时域响应的影响。这一设计使软件不仅能够展示现象，还具备一定的定量分析能力。

### 4.5 可视化与交互模块

软件采用 Matplotlib 构建图形界面，界面由速度场面板、涡量或迷宫面板、升力历史曲线面板及底部状态信息栏组成。使用者可通过键盘和鼠标对障碍物、参数和显示模式进行实时控制，从而提升实验灵活性和演示效果。软件同时支持保存静态截图以及导出 GIF、MP4 动画，方便形成实验报告和展示材料。

## 5. 核心技术原理

### 5.1 基于 D2Q9 的格子玻尔兹曼求解

软件采用二维九速度离散模型，将流体粒子分布函数定义在规则网格上，并通过“碰撞 + 迁移”的方式实现时间推进。程序先根据分布函数求取密度和速度，再构造平衡态分布函数，并以 BGK 单弛豫时间模型进行碰撞更新，随后沿离散速度方向完成迁移。这一方法具有结构清晰、易于离散和适合可视化展示等优点，非常适合用于教学和中等规模数值实验。

### 5.2 多边界条件处理机制

软件在左侧入口采用速度边界条件，在右侧出口采用零梯度边界条件，并支持上下边界周期性处理或反弹边界处理。在障碍物边界处，系统采用反弹边界规则，使得流体在固体表面满足无滑移条件。这种组合方式既保证了基础物理合理性，也兼顾了交互式实时演示的稳定性。

### 5.3 图像障碍导入与几何参数建模

除规则几何体外，软件能够将外部黑白图像缩放并映射到仿真网格中，以像素二值化结果直接构成障碍物掩码，实现复杂边界快速建模。此外，程序通过解析翼型厚度分布公式与几何旋转变换生成 NACA0012 翼型，有利于展示空气动力学中攻角变化带来的流场差异。

### 5.4 流体场与算法路径的耦合显示

软件不仅对迷宫图像执行 BFS 最短路径搜索，还将搜索结果与流体演化过程耦合。路径首先以逻辑结果形式存储，随后在仿真过程中依据局部速度是否达到阈值决定是否显示，从而实现路径随流动推进逐步显现的效果。这一机制使软件兼具数值模拟、图论算法展示和动态可视化表达三种特征。

### 5.5 三重气动力估算与涡核检测

软件从不同角度对障碍物所受作用力进行估算，包括基于边界分布函数反弹的动量交换法、基于局部压力和边界法向的压力积分法，以及基于环量估算的升力方法。同时，软件通过速度梯度张量构造 Q 准则，对旋涡核心区域进行检测和高亮显示，增强了对尾迹结构和升力变化关系的分析能力。

## 6. 软件主要特色

1. 将流体动力学数值模拟、迷宫求解、几何建模和可视化交互集成于同一软件系统中，功能复合度高。
2. 支持多种障碍物和多种展示模式，适用于课程实验、原理演示和成果展示等不同场景。
3. 采用矩阵化实现方式，提高了二维网格流体求解效率，降低了对高性能硬件的依赖。
4. 具备较强的软件展示性，既能体现算法逻辑，也能输出图像和动画成果，便于教学传播与申报使用。

## 7. 运行环境说明

本软件在 macOS 26.3 环境下完成开发与验证，实际开发与运行时使用 Conda `fai` 虚拟环境，Python 版本为 3.12.12。软件运行依赖的核心第三方库包括 NumPy 2.4.2、Matplotlib 3.10.8、Pillow 12.1.1 和 OpenCV 4.13.0，其中 OpenCV 用于外部图像障碍导入与迷宫边界构建功能。对于动画导出功能，若需导出 MP4 格式，可配合 FFmpeg 使用。软件对硬件要求较低，在具备多核处理器和 8GB 以上内存的个人计算机上即可稳定运行。

## 8. 适用场景

本软件适用于以下场景：

1. 流体动力学课程中的卡门涡街、尾迹分离、边界效应与升力演化教学演示。
2. 智能演化算法、图搜索算法与数值计算课程中的综合实验展示。
3. 航空航天基础课程中翼型绕流、攻角影响和升力变化的可视化分析。
4. 科研展示、课程答辩、竞赛汇报和科普演示中的图形化说明材料制作。
