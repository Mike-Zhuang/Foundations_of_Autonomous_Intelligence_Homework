## 推荐排版（对应老师建议）

1. 左侧放“左竖总览”（来自四联总图的左侧分区）。
2. 右侧从上到下放三个横向子图（A/B/C）。
3. 若一页信息过多，可将右下子图拆到下一页。

## 使用方式

1. 在 VS Code 打开上述 markdown 文件。
2. 使用 Mermaid 预览或复制 Mermaid 代码到在线 Mermaid Live Editor 导出 SVG。
3. 将 SVG 插入 PPT，保证放大后清晰。
