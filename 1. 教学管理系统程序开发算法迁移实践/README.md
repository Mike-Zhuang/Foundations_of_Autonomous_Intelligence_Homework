# 教学管理系统程序开发算法迁移实践

这是一个基于面向对象编程（OOP）实现的学生成绩管理与分析系统，支持多维度的成绩统计和图表可视化。

## 📌 功能特性
- **学生信息管理**：支持添加学生信息、成绩计算及按 GPA 排名。
- **成绩统计分析**：提供高等数学、英语、程序设计成绩分布以及总体绩点分布。
- **数据可视化**：自动生成详尽的可视化报告（成绩对比柱状图、平均绩点条形图、各分数段堆叠柱状图、前5强雷达图）。
- **数据导出**：一键将处理后的成绩单导出为 `.csv` 格式，方便在 Excel 中查阅。

## 🚀 基本操作与运行方法
1. **环境依赖**：
   本项目依赖 `matplotlib` 和 `numpy`。请确保运行前已安装相关库：
   ```bash
   pip install matplotlib numpy
   ```
2. **执行程序**：
   在本项目目录下，运行以下命令即可：
   ```bash
   python Manager_Class_Students.py
   ```
3. **输出结果**：
   - 终端中将会输出班级的初始化数据、整理后的数据及简易的文本分布条形图。
   - 目录中会自动生成 `student_scores.csv` 保存原始明细。
   - 自动弹出并保存包含四张子图的高级可视化报告图表 `student_scores_visualization.png`。
