# WIM（动态承重系统）数据分析

本项目用于分析 WIM（Weigh-In-Motion，动态称重系统）系统采集的车辆原始传感器数据，旨在深入挖掘交通流量、车重分布及时间维度上的流量变化规律。

## 📌 功能特性
- 车型分布统计（饼图）
- 各车道车型及载重分布统计（最大值、最小值、均值分析）
- 交通流量时间序列分析（24 小时流量时序点线图）
- 工作日与双休日车流及主要车型流量变化规律对比

## 🚀 基本操作与运行方法
1. **环境依赖**：
   本项目依赖 `matplotlib` 进行数据可视化，请前提安装相关库：
   ```bash
   pip install matplotlib
   ```
2. **准备数据**：
   请确保当前目录下已经存在核心数据文件 `WIMData.csv`。
3. **执行程序**：
   直接运行主程序脚本：
   ```bash
   python WIN.py
   ```
4. **输出结果**：
   此脚本以批量化处理的方式设计。它会在后台静默完成所有 6 个主干任务与拓展任务的计算，**在命令行中输出各项文本分析表格和数据比对结果**，并**在当前目录下自动保存超过 10 张高清图表（`.png` 格式）**。
   生成的图像包括但不限于：总体车型分布饼状图、每条车道的具体分布、轴重趋势点线图、固定时间流对比折线图等。
