# WIM（Weigh-In-Motion）数据分析 - 作业要求

## 项目概述

WIM（Weigh-In-Motion）是一种动态称重技术。该技术允许车辆在以正常行驶速度通过时测量其轴重和总重，而无需停车。目前广泛用于高速公路收费站、超载检测站、桥梁保护以及物流数据分析。

## 数据说明

现有 `WIMData.csv` 文件，包含了某大桥2024年3月份部分WIM数据。

### 数据字段

| 字段名称 | 代表含义 | 备注 |
|---------|---------|------|
| PassTime | 车辆通过时间 | 格式为（24小时制）：年-月-日 时:分:秒 |
| Lane | 车道编号 | - |
| AxleNum | 车辆的轴数 | - |
| TotalWeight | 车辆总重 | - |
| AxleWeight1~8 | 车辆轴重 | - |
| AxleDist1~7 | 车辆轴距 | 相邻两排轴之间的距离为轴距 |
| Speed | 通过WIM时的车速 | km/h |
| OverWeight | 是否超重 | 0：未超重；1：超重 |
| VehicleType | 车辆类型 | A：小客车；B：小型货车；C：大客车；D：中型货车；E：大型货车；F：特大型货车；G：铰接或拖挂汽车；I: 此车型不参与统计 |

## 任务要求

### 1. 所有车辆类型占比分析
完成如下分析：
- 统计所有车辆中，各类车的占比
- 绘制成饼图

**Deliverable：** 饼图（含占比数据）

---

### 2. 每个车道内车辆类型占比分析
完成如下分析：
- 统计每个车道中，各类车的占比
- 为每个车道绘制一个饼图
- **问题**：根据图中数据你能看出什么特点？

**Deliverable：** 多张饼图 + 特点描述

---

### 3. 方向对比分析
完成如下分析：
- 1~3车道为一个方向，4~6为另一个方向
- 统计两个方向上通过的车总数是否相同
- 统计每种车型的数量是否相同

**Deliverable：** 对比表格 + 数据分析

---

### 4. 车重规律分析（点线图）
完成如下分析：
- 通过每个车道的车重平均值、最大值、最小值
- 绘制成点线图
  - 横坐标：车道编号
  - 纵坐标：车重（g或kg）
- **问题**：
  - 你能看出什么规律？
  - 这个结果有何意义？

**Deliverable：** 点线图（包含平均值、最大值、最小值曲线）+ 规律总结

---

### 5. 车流量分析（点线图）
完成如下分析：
- 选择具有完整数据的某天（可以选择2024/03/05~2024/03/28中的某一天）
- 统计出每小时车流量（每小时通过WIM的车辆总数）
- 绘制成点线图
  - 横坐标：小时（0~23）
  - 纵坐标：车流量（车辆数）
  - 数据点：应包含24个点（每小时一个）
- **问题**：你能发现什么规律？

**Deliverable：** 点线图 + 规律发现

---

### 6. 工作日vs双休日对比分析（选做）
在第5题的基础上进一步分析：
- 对比工作日（周一~周五）和双休日（周六~周日）
- 分析一天当中车流随时间的变化规律是否有变化
- 进一步分析：如果分每种车型统计车流量，会有怎样的规律？

**Deliverable：** 对比图表 + 规律分析
